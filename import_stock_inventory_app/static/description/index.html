Import/Update Stock Inventory Adjustment

This odoo app is used to import picking/stock inventory adjustments with many options like file type,product,location and auto validation can be really helpful data for warehouse module in Odoo ERP. Using this app user can import opening stock balance of product with lot number which also has the option to create product automatically. We also provide option to import and update inventory adjsutment while importing stock also have option to auto validate inventory adjustment.

Import Inventory from XLS/CSV

Import stock inventory adjustments from xls/csv file.

Add/Update Inventory

User can easily add or update inventory adjustments.

Import Inventory Location

Options like location name, barcode and external id is provided to choose location for inventory adjustments.

Import Product

Import product for inventory adjustments from XLS/CSV file with product name, internal reference, barcode and external id options.

Create New Product

User can create a new product from XLS/CSV file.

Select Accounting Date

Select an accounting date for inventory adjustment.

Validate Stock Inventory

User can validate inventory adjustments imported from CSV/XLS file.

Skip Validation with Log

User can skip inventory validation and can get a log of incorrect data of file.

Warnings

If selected xls/csv file contains any wrong data then validation warnings are provided.

Example of XLS and CSV Formate

Make Your xls and csv files as per given example to import inventory adjustments.

  1. If you want to search product or location by name then in product and location column add product name and location name.
  2. If you want to search product or location by barcode then in product and location column add product barcode and location barcode.
  3. If you want to search product by internal reference then in product column add product internal reference.
  4. If you want to search product or location by external id then in product and location column add product external id and location external id.

Make Your xls and csv files as per given example If you have enable create product with import stock option.

If you select search product by name then follow given formate of example file.

If you select search product by barcode or internal reference or external id then provide product name on internal reference column.

Skip Validation Or Restrict Validation

If user select skip validation option and import file contains wrong data then wrong data will be skipped and remains correct data will imported from file, user can see validation logs for incorrect data.

If user select restrict with validation then if there is any wrong data on file then user can see the warnings for wrong data and it will be restricted to add or update inventory adjustment.
