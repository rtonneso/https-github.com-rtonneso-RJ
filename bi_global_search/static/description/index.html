Global Advance Search Odoo Apps

Global Advanced google like search in Odoo, this Odoo apps provide flexibility to search any record or model data according to the configured models and fields from the search box in the header menu. Now it is easy to find record and data in Odoo system by this Odoo module. This Odoo apps helps to do Global Search in complete Odoo with configurable option.

Features

Global Search Configuration

Configure global search with model and fields.

Search All Model Data

Automatically return search result for configured model data in global search view.

Global Search Access Rights
