Odoo 13 POS Gift Card

This module allow user to generate gift cards and customer can use this cards for make payment.

Let's see the video

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

How to use

  • Here we can use these features by configuration settings.


  • After enable the configuration there is one button, by clicking the Gift Card button there is open new screen, in this screen gift card can be create.


  • There are two type of cards, Free and Paid.
  • Here we can see the popup for create paid gift card in below image.


  • Here is confirmation popup before going to create paid card or recharge card.






  • Here we can see the popup for create free gift card in below image.




  • After customer can redeem gift card as per below images.






  • As per below image user can change expiry date of gift card.


  • As per below image user can recharge the gift card .


  • As per below image user can exchange the gift card number.


  • We can see card usage as per bellow image.
  • We can see card recharge history as per bellow image.
  • We can see card usage history by clicking " Gift Card > Used Card ".
  • We can see card recharge history by clicking " Gift Card > Recharged Card ".
  • We can define card type with name and code.
  • After using gift card we can get the email as per below image.
  • Every one month we get the email of our gift card balance.
  • When expire our gift card before 7 days we get email for renew card.
  • When change our gift card number then also we get the email.
